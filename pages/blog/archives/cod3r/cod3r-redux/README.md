## Redux Simples

### Rodar o Projeto 
 
Create the app in the development mode: `npx create-react-app redux-simples`: <br/>

Runs the app in the development mode: `npm start`: <br/>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
